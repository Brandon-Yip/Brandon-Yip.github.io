Edufy is a prototype platform designed to promote a loosely competitive educational experience for children.

With online testing websites being prominent in today's society, it is important to distinguish one from another. Edufy's goal is to encourage the competitive spirit in young students. The main leaderboard updates in real time after a top-scoring user finishes another quiz

  • Notable Features of Edufy

    Live Leaderboard + Profile Pictures

    Edufy's live leaderboard updates everytime a top user's score changes or a new user enters the top 10. Edufy dynamically generates a profile picture for the user using a third-party source and the account username.

  • Signup + cookies

    Edufy stores the user account data in a database hosted on Heroku. After logging in once, the user information will be stored in the browser's cookies system, meaning users' will have easy logins.

  • Quiz Format + Generation

    Edufy's quiz format allows for users to open and close questions as they please. This allows users to hide questions they are confident in. Quizzes are generated using imbeded html tags, heroku API calls, and displayed in a jsx format with bootstrap

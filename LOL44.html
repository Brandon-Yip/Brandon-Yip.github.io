LOL44

League of Legends 44 aka LOL44 is a tool catered toward the new players of the popular game League of Legends. The project aims to provide helpful champion suggestions based on the players current character preferences.

  • Notable Features of LOL44

    Accelerated loading times

    Load times are the worst and are especially present when dealing with foreign API's using special accessing rules. LOL44 Operates on pre-generated data (stored as a CSV file) from the RIOT API to minimize HTTP requests and maximize performance!

  • Champion Comparison Percentage

    Developed using "shortest path" graph algorithms, the user is able to compare any two recommended champions to see how similar the suggestions are to one another. The percentage is based off the initial character!

LOL44 can be split whollistically into two parts. The first is the upperlevel system that runs when the user is present. The second is the generation of the csv database (*ran only by us as developers or an automated system*) to limit users' loading times.

The second diagram below demonstrates the steps taken for the developer or automated system in order to generate an dataset for the character relationships.

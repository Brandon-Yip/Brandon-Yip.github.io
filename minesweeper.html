  • Notable Features of Minesweeper

    Custom level importing

    Using a txt file with 1's and 0's, the player can manually input their own custom levels. These levels can also be stored in the Test #1 - Test #3 buttons for quick access.

  • X-ray/Cheat mode

    The X-ray mode can be toggled for players who immediately want to see where all mines are on the map. The mode allows for players to continue in a stuck position.

  • Classic Features

    A flagging system, a mine counter, and the classic smiley face are all included in this version of minesweeper. Implemented using quickly changing png files within the SFML library.
